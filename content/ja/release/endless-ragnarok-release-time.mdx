<Section icon={Swords} title="新たなゲームシステム：「コンフラックス」と「マスタートレイト」">

**エンドレスラグナロクの配信時間**や価格設定だけでなく、この拡張コンテンツではゲームプレイを劇的に変化させる新システムが導入されます。最も重要な追加要素は、ソロプレイに特化したローグライトモード**「コンフラックス（The Conflux）」**です。このモードでは、ランダムなパズルや戦闘エンカウントが待ち受けるエリアを順番に攻略していきます。エリアをクリアするごとに、その挑戦中のみ有効な一時的なバフ（パーク）を選択し、通常ではあり得ないほど強力で実験的なキャラクタービルドを構築することができます。

さらに、**「マスタートレイト（Master Traits）」**システムにより、すべてのキャラクターに新たな成長要素が加わります。これらのトレイトを使用することで、お気に入りのキャラクターをこれまでの限界突破以上に強化し、純粋なアタッカー、不沈のタンク、あるいは究極のサポート役といった専門的なロールへ特化させることが可能になります。

**拡張コンテンツの主な特徴：**
- **コンフラックス**: 独自の宝箱報酬が手に入る、繰り返しプレイ可能なソロ向けエンドゲームモード。
- **マスタートレイト**: キャラクターの育成をさらに深め、エンドゲームビルドを極めるための新システム。
- **召喚システム**: 戦闘中にプロトバハムートなどの強力な存在を召喚して使役。
- **カオス（Chaos）クエスト**: 新たな敵タイプ「ラグナリア」が登場する最高難易度ティア。
- **プライマルバースト**: 特定の戦闘条件を満たすことで発動する、壊滅的な威力を誇るパーティ全員での連携攻撃。

| システム | 主な役割 | 最適なプレイヤー |
| :--- | :--- | :--- |
| **コンフラックス** | ソロ素材集め | シングルプレイでの進行を好むプレイヤー |
| **マスタートレイト** | パワー上限解放 | キャラクターの性能をレベル100以上の効率へ引き上げたい方 |
| **召喚** | 戦闘のユーティリティ | 緊急時の防御や、瞬間的な超大ダメージを狙う時 |
| **カオスクエスト** | エンドゲームの挑戦 | 究極の試練に挑みたいマルチプレイパーティ |
