<Callout type="info" title="クイックガイド">
- **最優先事項**: アップグレードする前に、公式の**エンドレスラグナロクの動作環境**でハードウェアの互換性を確認してください。
- **PCストレージ**: 最適なロード時間とパフォーマンスを得るために、SSDに**90 GB以上の空き容量**があることを確認してください。
- **必要メモリ（RAM）**: 標準（Standard）およびウルトラ（Ultra）グラフィック設定のどちらでも、最低**16 GBのRAM**が必要です。
- **コンソール版の容量**: プラットフォームによって異なり、**PS5で21 GB**、**PS4で11 GB**、**Switch 2で10 GB**が必要です。
- **目標パフォーマンス**: 推奨スペックを満たすことで、ウルトラ設定で**1080p解像度 / 60 FPS**を達成できます。
</Callout>

<Section icon={Shield} title="公式PC動作環境">

PC版『グランブルーファンタジー リリンク：エンドレスラグナロク』の膨大な追加コンテンツを楽しむには、システムが特定のハードウェア基準を満たしている必要があります。公式の**エンドレスラグナロクの動作環境**では、目標とする画質とフレームレートのパフォーマンスに応じて、2つの異なるハードウェア構成が定められています。

本作は64ビットのオペレーティングシステムとプロセッサを必要とし、2026年リリース版ではWindows 11（64ビット）が標準的な動作環境となります。以下は、最低スペックと推奨スペックの完全な内訳です。

| ハードウェア構成要素 | 最低動作環境 | 推奨動作環境 |
| :--- | :--- | :--- |
| **オペレーティングシステム (OS)** | Windows 10 / 11 (64-bit) | Windows 10 / 11 (64-bit) |
| **プロセッサ (CPU)** | Intel Core i3-9100 / AMD Ryzen 3 3200G | Intel Core i7-8700 / AMD Ryzen 5 3600 |
| **メモリ (RAM)** | 16 GB RAM | 16 GB RAM |
| **グラフィックカード (GPU)** | NVIDIA GTX 1060 6GB / AMD RX 580 8GB | NVIDIA RTX 2080 8GB / AMD RX 6700 XT 8GB |
| **DirectX バージョン** | バージョン 11 | バージョン 11 |
| **ディスク空き容量** | 90 GB 以上の空き容量 | 90 GB 以上の空き容量 |
| **ストレージタイプ** | SSD推奨 | SSD必須 |
| **目標パフォーマンス** | 1080p / 30 FPS（標準グラフィック） | 1080p / 60 FPS（ウルトラグラフィック） |

<Callout type="warning" title="SSDへのインストールを強く推奨">
ゲーム自体は通常のHDD（ハードディスクドライブ）でも動作する場合がありますが、本拡張コンテンツはSSDにインストールすることを強く推奨します。HDDでは、重度のアセット読み込み（ストリーミング）問題、激しい召喚演出時のカクつき（スタッター）、およびロード時間の長期化が発生する可能性があります。
</Callout>

</Section>

<Section icon={Sparkles} title="低スペックPC向けの最適化手順">

お使いのハードウェアが最低限の**エンドレスラグナロクの動作環境**の境界線上にある場合でも、ゲーム内の主要な設定を調整することで安定したパフォーマンスを得ることができます。画質を極端に犠牲にすることなくフレームレートを最大化するために、以下の手順に従ってください。

<Steps>
  <Step num="1" title="グラフィックプリセットの調整">
    メインメニューのグラフィック設定に移動し、全体のプリセットを**「標準（Standard）」**に変更します。これにより、影の品質やボリュメトリックフォグなど、負荷の高い設定が自動的に引き下げられます。
  </Step>
  <Step num="2" title="フレームレート上限の設定">
    GPUの性能に応じて、ゲーム内のフレームレート上限を**30 FPS**または**60 FPS**に設定します。フレームレートを無制限にすると、古いグラフィックカードでは深刻なサーマルスロットリング（熱による性能低下）が発生する可能性があります。
  </Step>
  <Step num="3" title="V-Syncとモーションブラーの削減">
    モーションブラーを完全にオフにして、処理のオーバーヘッドを削減します。画面の引き裂き（ティアリング）が発生する場合のみV-Sync（垂直同期）を有効にし、それ以外の場合は入力遅延を最小限に抑えるために無効にしておきます。
  </Step>
  <Step num="4" title="バックグラウンドアプリの最適化">
    ゲームを起動する前に、負荷の高いウェブブラウザ、オーバーレイ、録画ソフトなどを閉じてください。これにより、16 GBのシステムメモリ（RAM）すべてをゲームの描画に割り当てることができます。
  </Step>
</Steps>

<Callout type="success" title="召喚演出時のパフォーマンス">
新しく導入された「召喚（Summon）」システムは、非常に精細なビジュアルエフェクトとパーティクルシステムを使用しています。プライマルバースト（Primal Bursts）中にラグが発生する場合は、グラフィックメニューの**「VFX品質」**設定を下げることで、フレームレートが即座に安定します。
</Callout>

</Section>
